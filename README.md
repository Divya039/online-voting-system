# Online Voting System

A secure, web-based voting platform that allows users to cast votes remotely.

## Features
- Unique user authentication to prevent duplicate voting
- Role-based access for Admin and Voters
- Secure session management
- Real-time vote tracking

##  Technologies Used
- **Frontend:** HTML, CSS
- **Backend:** PHP
- **Database:** MySQL

##  How to Run
1. Clone the repo: `git clone https://github.com/Divya039/online-voting-system`
2. Import the SQL file into MySQL
3. Run on localhost using XAMPP
